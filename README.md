# AD-User-Automation
Automated user onboarding process, reducing manual setup time and ensuring consistency across multiple schools or offices

## Technologies Used
- PowerShell
- Active Directory
- CSV input automation

## How It Works
An administrator enters new user information into the NewADUsers.csv file.
When the script is executed, it reads the CSV data and automatically provisions user accounts in Active Directory based on the provided details.
- Creates user accounts
- Assigns group memberships
- Configures user attributes
- Enables mailbox provisioning
